Energy Drink

  • Steps to change old HTML to HTML5.
  1. The (doctype) element is just (!doctype html) now.
  2. The (meta) element is just (meta charset='utf-8').
  3. Remove the type attribute for the (link) element.
  4. Remove the type attribute for the (script) element.
  • Use var instead of int and string in javaScript.
  • Use document.write(lyrics) to write the variable 'lyrics' to the screen.
