Chapter 14: Web forms

Starbuzz Bean Machine
Starbuzz Bean Machine Table

Learned from Chapter 14:

  • How to use the (form) element to create forms.
  • How to use the action attribute inside the (form) element to specify the URL of the web server. Put the name of the web application at the end of the action attribute to process the form data.
  • How to use the method attribute to determine how the form data will be sent to the server. This includes POST.
  • How to use the (input) element in a form to create a one line control. The type attribute='text' creates this. The name attribute is used by the web application.
  • How to use the (input) element to create a button that allows you to submit a form. The type='submit' creates this.
  • How to use the (input) element in a form to create a single control with several buttons, with one being selected at a time. The type='radio' creates this. The name attribute must be the same. The value attribute must be different.
  • How to use the (input) element in a form to create a checkbox control that can be checked or unchecked. The type='checkbox' creates this. The name attribute must be the same. The value attribute must be different.
  • How to use the (textarea) element to create a multi-line text area that you can type into. The name attribute gives the element a unique name. The rows and columns attributes set how big the textarea is. Use (/textarea) to close.
  • How to use the (select) element to create a menu control in the web page. The name attribute names the select element.
  • How to use the (option) element inside the (select) element to list the possible values of the menu. Use the value attribute to name the options.
  • How to use the checked attribute to pre-select an input being checked.
  • POST packages up your form variables and sends them behind the scenes to your server. GET packages up your form variables but appends them on the end of the URL.
  • Use GET when people can bookmark a page.
  • Use POST for private information and if you have a (textarea) that sends a lot of data.
  • How to use the (fieldset) element to group common elements together. Use (legend) inside (fieldset) to provide a label for the group. Use (input) inside (fieldset) to list the values.
  • How to use the (label) element to provide more information about the structure of your page.
  • How to use the (input) element in a form to have a password field. Use type='password' to have a masked password.
  • How to use the (input) element in a form to send a file to a web application. Use type='file' and name='doc'.
  • How to use the (select) element in a form to pick multiple options. Use the multiple='multiple' attribute. Use option value to list each value you want to use.
